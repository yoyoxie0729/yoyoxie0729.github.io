开源博客 Hexo 部署过程全记录

2018-03-27
基于 Node.js 的开源博客软件 Hexo 部署流程 安装 HomeBrew $ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)" 安装 npm $ brew intall npm 安装 hexo-cli $ npm install -g hexo-cli 初始化 Hexo $ hexo init blog 生成第一篇博客 $ hexo n "Hello,Hexo!" 用 Markdown 语法编写博客 $ vim Hellp,Hexo!.md 生成静态资源并本地发布测试 $ hexo g $ hexo s 或 $ hexo s -g 访问 http://localhost:4000/ 查看部署成果 修改站点配置文件 _config.yml # Deployment ## Docs: https://hexo.io/docs/deployment.html deploy: type: git repo: https://github.com/yourname/yourname.github.io.git branch: master 发布博客 $ hexo clean $ hexo g $ hexo d Nice!
