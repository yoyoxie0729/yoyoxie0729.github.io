## 函数部分
### Oracle 11g版本`WMSYS.WM_CONCAT()`函数
```sql
-- 合并记录，逗号隔开
SELECT '字段1','字段2',WMSYS.WM_CONCAT('字段3')
  FROM '表1'
 GROUP BY '字段1','字段2'
 ORDER BY '字段1';
```
#### 效果如下
- 原表

| 字段1 | 字段2 | 字段3 |
| :---: | :---: | :---: |
|   A   |  A1   |  AA2  |
|   A   |  A1   |  AA3  |
|   B   |  B1   |  BB1  |
|   B   |  B1   |  BB2  |

- 结果表

| 字段1 | 字段2 | `WMSYS.WM_CONCAT(字段3)` |
| :---: | :---: | :----------------------: |
|   A   |  A1   |         AA2,AA3          |
|   B   |  B1   |         BB1,BB2          |
